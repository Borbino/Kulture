# 개인정보보호 및 GDPR 정책 (초안)

## 목적
Kulture 플랫폼은 글로벌 사용자(EU 포함)를 대상으로 하므로, GDPR 및 국내 개인정보보호법을 준수합니다.

## 수집하는 개인정보
- 필수: 이메일 주소, 닉네임
- 선택: 프로필 사진, 소셜 로그인 정보
- 자동: IP 주소, 쿠키, 접속 로그

## 개인정보 사용 목적
- 회원 가입·인증·로그인
- 콘텐츠 추천·맞춤형 서비스 제공
- 서비스 개선 및 통계 분석

## 개인정보 보관 기간
- 회원 탈퇴 시 즉시 삭제(법적 의무 제외)
- 법적 의무: 전자상거래법 등에 따라 최소 기간 보관

## 사용자 권리(GDPR)
- 열람권: 본인 정보 조회
- 삭제권: 본인 정보 삭제 요청
- 정정권: 잘못된 정보 수정 요청
- 이동권: 데이터 다운로드

## 쿠키 및 트래킹
- 필수 쿠키: 로그인 세션 유지
- 분석 쿠키: Google Analytics 등(사용자 동의 후 활성화)

## 제3자 제공
- 원칙: 사용자 동의 없이 제3자에게 제공하지 않음
- 예외: 법적 의무(법원 명령 등)

## 보안 조치
- 데이터 암호화(전송·저장)
- 접근 제어(최소 권한 원칙)
- 정기 보안 점검
