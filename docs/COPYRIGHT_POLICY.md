# 저작권 및 콘텐츠 정책 (초안)

## 목적
K-Culture 콘텐츠(드라마, K-Pop, 영화 등)를 공유하는 플랫폼으로서, 저작권을 존중하고 법적 리스크를 최소화합니다.

## 콘텐츠 유형
- 사용자 생성 콘텐츠(UGC): 리뷰, 댓글, 평점 등
- 외부 콘텐츠: 공식 포스터, 예고편(출처 명시 필수)

## 저작권 원칙
- 원칙: 저작권자의 허락 없이 원본 콘텐츠(영상, 음원 등) 업로드 금지
- 허용: 공식 임베드(YouTube, Spotify 등), 출처 명시한 포스터/스크린샷(Fair Use 범위)
- 금지: 불법 다운로드 링크, 무단 업로드, 저작권 침해 콘텐츠

## Fair Use(공정 이용)
- 비평·리뷰·교육 목적의 짧은 인용은 허용
- 원본 콘텐츠를 대체하지 않는 범위

## 신고 절차(DMCA)
1. 저작권자가 침해 신고 접수
2. 운영팀이 24시간 내 검토
3. 침해 확인 시 즉시 삭제 및 사용자 경고
4. 반복 위반 시 계정 정지

## 사용자 책임
- 사용자가 업로드한 콘텐츠의 저작권은 사용자에게 책임
- 침해 콘텐츠 업로드 시 법적 책임은 사용자에게

## 플랫폼 면책
- 플랫폼은 사용자 콘텐츠에 대해 사전 검토 의무 없음(단, 신고 시 즉시 조치)
