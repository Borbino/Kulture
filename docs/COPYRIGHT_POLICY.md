## 1. 정책 목적 및 적용 범위

### 목적

Kulture는 K-Culture 콘텐츠(K-Pop, K-Drama, K-Beauty, K-Food 등)를 전 세계 팬과 공유하는 글로벌 커뮤니티 플랫폼입니다. 본 정책은 다음을 보장합니다:

- 원저작자의 저작권 완전 존중
- DMCA(미국 디지털 밀레니엄 저작권법) 및 한국 저작권법 준수
- AI 생성 2차 창작물의 법적 안전성 확보
- 플랫폼 이용자 보호

### 적용 범위

| 대상 | 적용 여부 | 비고 |
| -------- | --------- | ---- |
| 플랫폼 자체 제작 기사/분석 | ✅ 적용 | Kulture 저작권 귀속 |
| AI 보조 생성 콘텐츠 | ✅ 적용 | lib/copyrightChecker.js 검증 필수 |
| 사용자 업로드 콘텐츠(UGC) | ✅ 적용 | 사용자 책임 원칙 |
| 외부 RSS 수집 콘텐츠 | ✅ 적용 | 요약·링크만 허용 |
| 공식 임베드(YouTube, Spotify) | ⚠️ 조건부 | 플랫폼 ToS 준수 시 허용 |

## 3. 공정 이용(Fair Use) 기준

### 미국 저작권법 107조 / 한국 저작권법 28조 기준

공정 이용은 다음 4가지 요소를 종합 판단합니다:

1. **이용 목적**: 비영리·교육·비평·뉴스 목적 ✅ / 영리적 상업 이용 ❌
2. **원저작물 성격**: 사실적 자료 ✅ / 극히 창작성 높은 예술 작품 ⚠️
3. **인용 분량**: 원문의 10~30% 이하 ✅ / 50% 이상 ❌
4. **시장 대체성**: 원작 시장에 영향 없음 ✅ / 원작 구매 필요성 감소 ❌

### Kulture 공정 이용 운영 기준

```
✅ 허용: 뮤직비디오 속 장면 1-3컷 + 분석 텍스트 (비평)
✅ 허용: 가사 일부 (1절 이하) + 해석 및 문화적 설명 (교육)
✅ 허용: 공식 발표 수치 인용 + 출처 명시 (뉴스 보도)
⚠️ 주의: 앨범 전체 가사 (저작권자 허락 필요)
❌ 금지: 영상/음원 전체 무단 게재
❌ 금지: 고해상도 공식 사진 무제한 사용
```

### 자동화 검증 (lib/copyrightChecker.js)

Kulture 플랫폼은 모든 AI 생성 콘텐츠에 대해 발행 전 자동으로 공정 이용 요건을 검증합니다:

- **직접 복사 감지**: 원본 대비 30% 이상 유사 텍스트 자동 차단
- **인용 길이 제한**: 단일 인용 300자 초과 시 경고
- **출처 표기 자동 부착**: 참고 자료가 있는 경우 자동 생성
- **Fair Use 지시자 검사**: 비평/분석 표현 최소 3개 이상 요구

## 5. AI 생성 콘텐츠 저작권

### 생성 원칙

Kulture의 AI 콘텐츠 생성 시스템(`lib/advancedContentGeneration.js`)은 다음 원칙을 준수합니다:

1. **원본 대체 금지**: AI 생성 기사는 원본 뉴스/콘텐츠를 대체하지 않고, 독자적 분석·해설을 제공
2. **2차 창작 표기**: 모든 AI 생성 콘텐츠에 `[AI 보조 작성]` 태그 자동 부착
3. **출처 표기 의무**: 참조한 RSS 피드, 원문 기사 링크 자동 생성
4. **저작권 고지**: `© [년도] Kulture / Kulture 편집팀 (AI 보조 작성). CC BY-SA 4.0` 형식으로 발행

### AI 생성 콘텐츠 발행 전 심사 기준

| 검사 항목 | 기준 | 처리 |
| -------- | ---- | ---- |
| 직접 복사율 | 30% 미만 | 초과 시 자동 차단 |
| 공정 이용 지시자 | 3개 이상 | 미달 시 수동 검토 |
| 출처 표기 | 1개 이상 | 미포함 시 자동 생성 |
| 품질 점수 | 65점 이상 | 미달 시 반려 |
| 표절 유사도 | 70% 미만 | 초과 시 자동 차단 |

### 소유권

- AI 생성 콘텐츠의 최종 편집물에 대한 저작권: **Kulture 귀속**
- AI 모델 자체 출력물: 공공 도메인 (현행법상 AI는 저작자 불인정)
- 참조 원본 콘텐츠: **원저작자 귀속** (Kulture는 2차 창작자 지위)

## 6. DMCA 신고 및 처리 절차

### 신고 접수

**이메일**: `copyright@kulture.wiki`  
**대응 언어**: 한국어, 영어  
**처리 시간**: 접수 후 **24시간 이내** 검토, **72시간 이내** 조치 완료

### 신고 시 필수 제출 정보 (DMCA 512조 요건)

1. **저작권자 정보**: 이름, 주소, 전화, 이메일
2. **원본 저작물 식별**: 원본 콘텐츠의 URL 또는 설명
3. **침해 콘텐츠 위치**: Kulture 내 침해 콘텐츠의 정확한 URL
4. **선의 진술**: "본인은 해당 이용이 저작권자, 그 대리인 또는 법률로 허가되지 않은 것으로 선의(good faith)로 믿습니다."
5. **정확성 선언**: "본 신고 내용이 정확하며, 나는 저작권자이거나 저작권자를 대리할 권한이 있습니다."
6. **서명**: 전자서명 또는 실명 서명

### 처리 단계

```
1단계: 신고 접수 확인 이메일 발송 (1시간 이내)
2단계: 콘텐츠 검토 및 침해 여부 판단 (24시간 이내)
3단계-A: [침해 확인] 즉시 삭제 + 업로더/생성자 경고 발송
3단계-B: [침해 불명확] 추가 정보 요청 또는 법무 검토 의뢰
4단계: 삭제 조치 결과 신고자에게 통보 (72시간 이내)
5단계: 이의제기(Counter DMCA) 접수 시 14일 유예 후 처리
```

### 허위 신고 경고

허위 DMCA 신고는 DMCA 512(f)에 따라 **법적 손해배상 책임**이 발생할 수 있습니다.

## 7. 사용자 콘텐츠(UGC) 정책

### 업로드 제한

사용자는 다음 콘텐츠를 업로드할 수 없습니다:

- 타인의 저작물을 무단 복제한 텍스트, 이미지, 영상
- 저작권 보호 음원 (배경음악 포함)
- 워터마크가 있거나 `© All Rights Reserved` 표시된 이미지
- 불법 다운로드 경로로 취득한 콘텐츠

### 사용자 책임 원칙

- 업로드 콘텐츠에 대한 저작권 책임은 **전적으로 업로더에게 귀속**됩니다.
- Kulture는 업로드 전 사전 규제 의무가 없으나, 신고 접수 즉시 조치합니다.
- 반복 침해자에게는 계정 정지 및 영구 이용 정지가 적용됩니다.

### 사용자 콘텐츠 라이선스 부여

사용자가 Kulture에 콘텐츠를 게시함으로써, Kulture에게 다음의 비독점적 라이선스를 부여합니다:

- 플랫폼 내 콘텐츠 표시, 배포, 번역(요약) 권한
- 추천/검색 알고리즘 활용 권한
- **제3자 상업적 이용은 불포함** (사전 동의 필요)

## 8. 저작권 침해 경고 단계

| 단계 | 사유 | 조치 |
| ---- | ---- | ---- |
| 1차 경고 | 최초 위반 (경미) | 콘텐츠 삭제 + 이메일 경고 |
| 2차 경고 | 반복 위반 | 콘텐츠 삭제 + 게시 기능 7일 정지 |
| 3차 경고 | 3회 이상 반복 침해 | 계정 30일 정지 + 법적 조치 검토 |
| 영구 정지 | 악의적 대규모 침해 | 계정 영구 정지 + 법적 책임 추구 |

**항소(Appeal) 절차**: 조치가 부당하다고 판단되는 경우, 30일 이내에 `appeal@kulture.wiki`로 이의제기 가능.

## 9. 크롤링 및 RSS 수집 정책

자세한 내용은 [CRAWLER_POLICY.md](./CRAWLER_POLICY.md) 참조.

### 핵심 원칙 요약

- **공개 RSS 피드만** 사용 (API 키 불필요 소스 우선)
- robots.txt 준수 (모든 수집 소스 사전 확인)
- 원본 기사 전문 복사 **금지** — 제목 + 요약 + 원문 링크만 허용
- 수집 데이터 기반 AI 생성 콘텐츠는 **반드시 `lib/copyrightChecker.js` 통과 후** 발행
- 소스 신뢰도(`SOURCE_RELIABILITY`) 점수 5점 미만 출처 콘텐츠는 수동 검토 필수
